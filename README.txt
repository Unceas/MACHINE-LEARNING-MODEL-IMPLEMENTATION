# CODTECH Internship Task 4: Spam Email Classifier

## Objective:
Build a machine learning model using scikit-learn to classify SMS messages as spam or ham.

## Files Included:
- Spam_Email_Classifier_Task4.py: Python script
- Spam_Email_Classifier_Task4.ipynb: Jupyter notebook version
- spam.csv: Sample dataset with 'v1' (label), 'v2' (message)
- README.txt: This file

## How to Run:
1. Make sure Python and scikit-learn are installed.
2. Run `Spam_Email_Classifier_Task4.py` using Python.
3. Alternatively, open `Spam_Email_Classifier_Task4.ipynb` in Jupyter.

## Output:
The script prints the model accuracy based on test data.
